## Phase 1: Codebox revision in scratch patch

**Purpose**: Replace scratch patch codebox with production-quality field math. Validate all topology modes and modulation before writing any patcher JSON.

⚠️ CRITICAL: No build script work until this phase is complete and all acceptance criteria pass.

- [ ] T001 Update `vortex_pix` codebox in `vortex-scratch.maxpat`: replace `1/(1+r*f)` falloff with `exp(-r * max(falloff, 0.0))` — clamp falloff to prevent negative values
- [ ] T002 Add modulation inlet sampling to `vortex_pix` codebox: declare `Param cx_amt(0.0)`, `cy_amt(0.0)`, `convergence_amt(0.0)`, `curl_amt(0.0)`; sample `in2`–`in5` as `(sample(inN, norm).x - 0.5) * 2.0 * amt` and add to respective params
- [ ] T003 Add `Param bypass(0.0)` to `vortex_pix` codebox; when bypass active output `vec(0.5, 0.5, 0.5, 1.0)` neutral field
- [ ] T004 Wire modulation inlets in scratch patch: add `in 2`, `in 3`, `in 4`, `in 5` objects in gen subpatcher; connect to codebox inlets; update `numinlets` on outer `vortex_pix` object to 5
- [ ] T005 Verify sink topology: convergence=0.5, curl=0.0 — smooth radial field pointing inward from all directions
- [ ] T006 Verify source topology: convergence=-0.5, curl=0.0 — field points outward cleanly, no artifacts. Document result in spec open question.
- [ ] T007 Verify center topology: convergence=0.0, curl=0.5 — pure rotation, no radial component
- [ ] T008 Verify spiral topologies: convergence=0.5 + curl=0.5 (spiral sink); convergence=-0.5 + curl=0.5 (spiral source)
- [ ] T009 Verify falloff: falloff=0 produces uniform field across frame; falloff=5 concentrates field tightly near fixed point; no ring artifact at any positive value
- [ ] T010 Verify cx/cy: move fixed point to corners and edges — field topology follows correctly
- [ ] T011 Verify bypass: output is flat 0.5/0.5/0.5/1.0 with no field structure visible in consumer_pix
- [ ] T012 Verify modulation: connect a WFG to a modulation inlet with amt > 0 — param offsets visibly and expressively
- [ ] T013 Calibrate defaults: confirm falloff=2.0 feels right; adjust if needed and record final default in spec

**Checkpoint**: All topology modes clean. Falloff has no ring artifact. Modulation inlets offset params. Bypass outputs neutral field. Defaults recorded.
